# 月配息基金循環轉換研究筆記

線上工具：https://fund-opal.vercel.app/
程式碼：本repo根目錄（index.html / app.js / engine.js / data.js）是部署給 Vercel 的靜態網站。
這個 `research/` 資料夾是背後的原始資料與探索用 Python 腳本，不會被部署，純粹留底以後續接續分析。

## 涉及的基金

| 簡稱 | 代碼 | 全名 | 計價幣別 |
|---|---|---|---|
| 摩根多重收益 | JFZN3 | 摩根投資基金-多重收益基金A股(美元對沖)(穩定月配) | USD |
| 安聯全球 | TLZN0 | 安聯全球永續發展基金-AMg穩定月收總收益類股(美元) | USD |
| 聯博美國成長 | ALBT8 | 聯博-美國成長基金AP總報酬月配美元 | USD |
| 安聯台灣科技 | ACDD04 | 安聯台灣科技基金（境內，配息可轉入此基金加碼） | TWD |

資料來源：moneydj.com 基智網，配息頁 `funddj/yp/wb05.djhtm?a=<code>`、淨值頁 `funddj/yp/yp010000.djhtm` 或 `yp011001.djhtm`。
這些頁面是 JS 動態頁，WebFetch 抓不全，目前資料都是請使用者另外用網站AI爬蟲、手動複製貼上取得，
範圍 2024/06/18 ~ 2026/06/17（約23個月，受限於聯博AP級別淨值資料只回溯到2024/07/24）。

## 檔案說明

- `*_div.csv`：各基金配息歷史（配息基準日、除息日、發放日、每單位配息金額）
- `*_nav.csv`：各基金每日淨值歷史
- `analyze.py`：原型腳本，算「除息後3天淨值表現」+ 基本的100萬台幣三基金循環轉換模擬（對應 webapp 的 engine.js 邏輯前身）
- `sensitivity.py`：敏感度分析 —— 轉換延遲天數、贖回款項日數對能否趕上下一檔配息的影響；配息轉出比例對本金侵蝕的影響
- `decay_projection.py`：用68次轉換的實際本金市值做回歸，外推本金「趨近於0」大概要幾年
- `leg_gaps.py`：算三條轉換腿（JFZN3→TLZN0、TLZN0→ALBT8、ALBT8→JFZN3）各自的緩衝天數
- `export_data_js.py`：把所有 CSV 轉成 `webapp/data.js`（網站讀取的內嵌資料）。改了CSV後要重跑這支才會反映到網站上。
- `output.txt` / `sensitivity_output.txt`：上面腳本的執行結果存檔

## 目前最重要的幾個結論

1. **三基金循環轉換的本金本身是淨虧損的**：2024/07/24~2026/06/17，TWD 100萬投入，本金期末市值剩70.3萬（侵蝕約30%），全靠累計配息49.1萬把總報酬拉回+19.46%。回歸分析估算：若這個趨勢線性外推，本金大概1.7年內剩一半、10年內剩零頭——但這只是外推，不是保證，這段期間剛好包含2025/04關稅衝擊的大跌，不能直接當預言。

2. **單純買進持有單一基金，本金侵蝕反而比三基金循環更輕**（98.7萬/97.8萬 vs 70.3萬），但三種策略的總報酬率其實很接近(19~22%)。代表循環轉換沒有真的提升報酬，只是讓本金數字更難看，這個對比對客戶很有說服力。

3. **保單條款的實際轉換時間差是真風險**：依保單(VLBY1-C01)附表四，海外基金轉出要等「基準日次一資產評價日」+「贖回款項日數」才會真正轉入下一基金。三條轉換腿的緩衝天數：JFZN3→TLZN0 只有2~6天（中位數4天），TLZN0→ALBT8 有9~15天，ALBT8→JFZN3 有7~11天。**JFZN3→TLZN0這一腿是死結**——如果真實的贖回款項日數超過2天（業界常見3~7天），這個策略大概率會持續摃龜，趕不上安聯全球的配息基準日。贖回款項日數的實際數值需要去查安聯人壽官網公告，這份保單條款裡沒寫。

4. **配息轉出比例需要多少才能讓本金不被侵蝕**：精算約19.11%（轉出到安聯台灣科技）。**但這個數字完全是安聯台灣科技這23個月剛好漲了3.8倍（年化近95%）才成立的**，換一段市況、換一支接收標的，這個門檻會完全不同，甚至可能不存在解（如果接收標的同期也在跌）。不能把19.11%當成一個可重複使用的法則。

5. **加保單貸款（借50萬@3.99%年息，單利）的試算**：在這段回測期間是正槓桿（策略報酬40.69% >> 貸款成本約7.6%/23個月），淨多賺16.56萬，把自己100萬的報酬率從40.69%推到57.24%。但保單條款規定借款本息超過帳戶價值60%/70%會被強制扣抵，如果本金又持續侵蝕、貸款利息又持續累積，兩個方向同時走壞，可能比想像中更快觸發。

## 還沒做、下次可以接續的方向

- **2022年的對照**：2022年是全球升息年，債券和成長股/科技股難得同時下跌，跟2024-2026這段（安聯科技暴漲）情況完全相反。目前**完全沒有2022年的任何資料**，需要重新跑一次跟前面一樣的資料蒐集流程（向 moneydj 抓2022年區間的配息+淨值）才能算出真實數字，不要憑印象推算。
- **標普500基金**：使用者問過用標普500當配息轉出標的，目前完全沒有對應的基金代碼或資料，需要先確認要追蹤哪一支標普500連結基金（境內/境外、避險/不避險）。
- 贖回款項日數的真實數值（查安聯人壽官網的「贖回款項日數」公告），目前 webapp 裡是讓使用者自己填一個假設值。
